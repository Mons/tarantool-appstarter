# tarantool-appstarter
Tarantool application starter

### Requirements
* tarantool >= 1.6
* luarocks
  - luafilesystem

### Run
Clone and run
```
bin/tntstartapp /path/to/app
```

You can add `bin/` folder to your `PATH` variable to run in arbitrary directory:
```
cd /path/to/my/app
tntstartapp .
```
